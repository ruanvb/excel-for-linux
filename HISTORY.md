# History

Excel for Linux was created to provide Linux users with a native desktop experience
for Microsoft Excel Online (Office 365).

Microsoft does not officially support a native Excel desktop application on Linux.
The only available route is to use the [Microsoft Excel web app](https://www.office.com/launch/excel)
through a browser. This project wraps that web app using Electron to deliver a
more integrated desktop experience on Linux.

This project is inspired by similar wrappers in the open-source community and
was built to fill the gap for spreadsheet users on Linux.

Contributions — whether through issues, pull requests, or ideas —
are always welcome and appreciated.
